# Loan Default Analysis & Insights

## Problem Statement
Understand which borrowers are more likely to default on loans and provide actionable recommendations.

## Dataset
Loan Default Dataset (~3.75MB) with borrower demographics and loan info.

## Tools
- SQL / MySQL Workbench
- Power BI
- (Optional) Python / Jupyter Notebook

## Approach
1. Data cleaning & preprocessing
2. Exploratory analysis to identify default patterns
3. Visualization of key factors
4. Insights and recommendations

## Key Insights
- Low-income borrowers (<30k) have highest default rate
- Self-employed borrowers are riskier
- Age less significant but young borrowers slightly higher risk

## Recommendations
1. Credit approval adjustments for risky segments
2. Early intervention and financial literacy support
